PREFAZIONE--------------------------------------------------------------
entrare in vs code e fare npm install nel percorso di hivemind per installare tutte le dipendenze


BACKEND-----------------------------------------------------------------
Creare un nuovo database hivemind in postgres
incollare il contenuto di tabelle.sql 
in vs code, entrare nella cartella server e nel file database.js cambiare le mie credenziali con le proprie



FUNZIONAMENTO--------------------------------------------------------------
In vs code entrare nella directory hivemind e fare npm run dev per avviare il front end e poi spostarsi in server e fare npm run dev per avviare il backend
